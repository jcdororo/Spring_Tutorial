chap02  
스프링의 핵심기능은 객체를 생성하고 초기화 하는것이다.
싱글톤 객체 : 별도 설정을 하지 않을 경우 스프링은 한 개의 빈 객체만을 생성하며, 이때 빈 객체는 "싱글톤 범위를 갖는다'고 표현한다.
싱글톤은 단일 객체를 의미하는 단어로서 스프링은 기본적으로 한 개의 @Bean 애노테이션에 대해 한 개의 빈 객체를 생성한다.
